# Figma Keyboard Shortcuts

- `SHIFT + 2` - zoom into the element
- `OPTION + drag` - copy an element
- `CMD + C then CMD + V` - copy paste
- `CMD + D` - duplicate
- `+/-` - Zoom in/out 
- `o` - Ellipse tool
- `keep SHIFT pressed` - to maintain constraints for images, shapes, etc.
- `CMD + G` - group selection
